# スワイショウ（腕振り運動）

## 📌 概要
スワイショウは、シンプルながら非常に効果の高い中国伝統の気功法の一つです。力を抜いて腕を前後に振るだけで、気の巡りを良くし、自律神経を整えるとされています。

## ✨ 効果
1. **自律神経の調整**: リズミカルな動きがセロトニンを分泌させ、精神を安定させます。
2. **肩甲骨周りの緩和**: デスクワークで固まった肩・首周りの血流を劇的に改善します。
3. **邪気の排出**: 力を抜いて振ることで、体内に溜まったストレスや停滞したエネルギーをリセットします。
4. **自然治癒力の向上**: 深い呼吸と共に繰り返すことで、身体の基盤を整えます。

## 🛠️ やり方
1. **立ち方**: 足を肩幅に開き、膝を軽く緩めます。背筋を伸ばし、顎を軽く引きます。
2. **脱力**: 肩から指先まで、完全に力を抜きます。
3. **振る**: 
    - 腕を前方へ肩の高さまで振り上げ（自然に）、その後、重力に任せて後ろへ振り切ります。
    - これを一定のリズムで繰り返します。
4. **回数・時間**: 5分〜10分（回数でいうと200回〜500回程度）から始めるのが理想的です。
