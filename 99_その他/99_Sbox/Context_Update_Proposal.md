# Context Integration Proposal (Safety First)

## 1. Update Proposal for: 00_マスター(Master_Context).md
### [NEW] 追加される情報
*   **Mission & Vision**: 
    *   **知識のエンタメ化 (Edutainment)**: 堅苦しく難解なテーマ（投資、数学、物理など）をマンガで親しみやすく変換する。
    *   **IPのマルチバース展開**: 固定キャラ（レミ、優斗等）を軸に多岐にわたる世界観（学園、SF、ビジネス）を展開。

## 2. Update Proposal for: 01_価値観(Core_Values).md
### [NEW] 追加される情報
*   **信念 (Visual First)**: 言葉の説明よりも「絵で見せる」ことを最優先する。
*   **判断基準 (わかりやすさとキャラの魅力)**: 正確すぎる説明より、キャラ（特にレミ）がいかに魅力的に解説できているかを重視。
*   **スタイル (Adaptability & Consistency)**: 画風は柔軟に変更するが、キャラの記号的特徴（レミの銀髪・赤スーツ等）は徹底維持する。

## 3. Update Proposal for: 03_執筆スタイル(Style_Guidelines).md
### [NEW] 追加される情報
*   **語り口 (Persona)**: 「敏腕編集者兼プロデューサー」。具体的かつ明確なディレクションを行い、曖昧さを排除する。
*   **推奨ワード**: 「元気 (genki)」「対決」「開拓」など、ポジティブでエネルギーのある言葉を好む。
*   **構成ルール**: 脚本やプロンプトから入り、高速でプロトタイプを修正する「プロトタイピング思考」。

## 4. Update Proposal for: 04_スキルとノウハウ(Skills).md
