# 🎬 映像にドラマを宿す！演出技法の基礎

ドラマとは、「人を引き込む力」。技術だけでなく、感情を揺さぶり、物語に没入させるための演出テクニックです。

## 📖 ストーリー構成の基本

### 1. 三幕構成 (Three-Act Structure)

ハリウッド映画の王道構造。物語を3つの段階に分ける。

| 幕 | 名称 | 内容 | 時間配分 |
| :--- | :--- | :--- | :--- |
| **第一幕** | セットアップ（設定） | 主人公の日常、世界観の提示、事件の発生（インシデント）。 | 全体の25% |
| **第二幕** | コンフロンテーション（対立） | 困難に立ち向かう、試練の連続、中間点での大きな転換。最も長い。 | 全体の50% |
| **第三幕** | レゾリューション（解決） | クライマックス（最大の山場）、解決、エンディング。 | 全体の25% |

### 2. 起承転結

日本の伝統的な物語構造。4つの段階で展開。

1. **起**: 物語の始まり、状況説明。
2. **承**: 展開、話が進む。
3. **転**: 予想外の展開、大きな変化。
4. **結**: 結末、まとめ。

### 3. ヒーローズ・ジャーニー（英雄の旅）

神話学者ジョセフ・キャンベルが提唱した物語の原型。

- 日常世界 → 冒険への召命 → 試練 → 報酬 → 帰還

## ⚡ 緊張感（サスペンス）の作り方

1. **情報の非対称性**
    - 観客だけが知っている情報がある（例：爆弾が仕掛けられているのを主人公は知らない）。
    - 「どうなるんだ！？」というハラハラ感を生む。
2. **タイムリミット（時間制限）**
    - 「あと5分で爆発する」など、時間の切迫感。
3. **予測と裏切り**
    - 観客の予想を少し裏切る展開で、飽きさせない。
4. **沈黙と静寂**
    - 音を消すことで、次に何が起こるかわからない不安を煽る。

## 💔 コンフリクト（対立・葛藤）

ドラマの核心は「対立」。対立がないとドラマは生まれない。

| 対立の種類 | 説明 | 例 |
| :--- | :--- | :--- |
| **人 vs 人** | キャラクター同士の対立。 | 主人公 vs 敵（悪役） |
| **人 vs 自己** | 内面的な葛藤、自分との戦い。 | 復讐すべきか、許すべきか |
| **人 vs 社会** | 社会や組織との対立。 | 巨大企業との戦い |
| **人 vs 自然** | 災害、環境との戦い。 | サバイバル、遭難 |
| **人 vs 運命** | 避けられない運命との対峙。 | 予言、時間制限 |

## 🎭 感情表現のテクニック

### 1. ショットサイズとアングル

- **クローズアップ**: 感情を強調（涙、笑顔、恐怖の表情）。
- **ローアングル**: 力強さ、威圧感。
- **ハイアングル**: 弱さ、孤独、無力感。

### 2. カット割り

- **カットバック**: 2つの場所を交互に見せて、緊張感を高める（追う者と追われる者）。
- **モンタージュ**: 短いカットを連続させて、時間経過や感情の高まりを表現。

### 3. 音楽と効果音

- **BGMの有無**: 静寂は緊張を、音楽は感情を増幅する。
- **ディエジェティック音**: 画面内の音（ラジオ、足音）。
- **ノンディエジェティック音**: 画面外の音（BGM、ナレーション）。

### 4. 照明と色

- **暗く青い光**: 悲しみ、孤独、夜。
- **温かいオレンジの光**: 安心感、日常、温もり。
- **赤**: 危険、情熱、怒り。

## ⏱️ ペーシング（テンポとリズム）

映像のスピード感をコントロールすることで、観客の感情を誘導する。

- **スローペース**: 静かなシーン、感情的な会話、余韻を残す。
- **ファストペース**: アクションシーン、追跡劇、クライマックス。
- **緩急のメリハリ**: 静と動を組み合わせることで、映像にリズムが生まれる。

## 🔑 ドラマを高める演出のコツ

1. **「Show, Don't Tell（見せろ、語るな）」**
    - セリフで説明するのではなく、映像で見せる。
    - 例：「彼は怒っている」と言うのではなく、拳を握りしめる映像を見せる。
2. **伏線と回収**
    - 序盤に何気なく見せたものが、後半で重要な意味を持つ。
3. **キャラクターの成長**
    - 主人公が旅を通じて変化する（弱かった人が強くなる、閉じていた心が開く）。
4. **視覚的メタファー（象徴）**
    - 映像に象徴的な意味を持たせる（雨＝悲しみ、桜＝別れ、など）。

> [!TIP]
> どんなに技術が優れていても、「人の心を動かす」ことがドラマの本質です。まずは「誰に、何を伝えたいか？」を明確にしましょう。

## 💪 アイディアが訴える力を強くする方法

実践的なドラマ構築のための具体的テクニック集。

### ① 物語のつかみを探す

- **物語の魅力はなにか、その話を独創的にしているものは何か？**
- 冒頭で視聴者の心を掴む「フック」を明確にする。

### ② 各場・人物が発露する最悪の出来事

- 各場・人物が地獄の様な目にあって生還するならハイコンセプトになる。
- 主人公に試練を与えることで、物語に深みが生まれる。

### ③ 対照的な登場人物

- **正反対の二人を一緒に仕事をさせる、旅に出させる。**
- 対照的な性格や価値観のキャラクターを配置することで、ドラマが生まれる。

### ④ 対照的な登場人物と環境

- 各場・人物と舞台を対照的にする。
- 例：富裕層のキャラクターを貧しい環境に置く、など。

### ⑤ アイディアをもう一つ足す

- できたアイディアにもう一つ捻りを加える。
- 二重構造、サブプロット、意外な展開を追加する。

### ⑥ 常套的な要素を変える

- 野球を下敷きにサイボーグ2007（SF要素を追加）。
- ありきたりなプロットを逆転させる。

### ⑦ ありきたりなプロットを逆転させる

- 夫が誘拐される、妻が夫を救って！
- 常識を覆す設定で新鮮さを生む。

### ⑧ さっかけとなる事件を面白くする

- もしも大統領が誘拐されたら...
- 強烈なインシデントで物語を始める。

### ⑨ 極端にしてみる

- **最高、最大、最多、最悪、最凶の同志人。**
- スケールを大きくすることで、インパクトを増す。

### ⑩ 時間に制限をつける

- **○○キロをしたまわると爆発（スピード）**
- タイムリミットは緊張感を生む最強の武器。

### ⑪ 舞台を強調する、舞台裏をみせる

- **誰も観たことの無い舞台**
- 特殊な舞台設定で、視覚的な興味を引く。

### ⑫ コンセプトそのものをフレンマにする

- 母が生き残る為に双子の一人を選ぶ
- 道徳的ジレンマ、究極の選択で観客の心を揺さぶる。

> [!NOTE]
> これらのテクニックは単独でも効果的ですが、**複数を組み合わせる**ことで、より強力なドラマを生み出すことができます。
