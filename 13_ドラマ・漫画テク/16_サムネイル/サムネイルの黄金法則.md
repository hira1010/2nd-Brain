# 🖼️ クリックされるサムネイルの黄金法則 (YouTube/SNS)

サムネイルは動画の「顔」。中身が良くてもサムネが悪ければ再生されません。

## 🎯 視線誘導の基本テクニック

1. **Z型の法則**: 人の視線は **左上 → 右上 → 左下 → 右下** （Zの字）に移動する。
    * **左上**: 「一番伝えたいキーワード」や「強烈なインパクト」を置く。
    * **右下**: 「オチ」や「続きが気になる要素」を置く。
2. **人の顔**: 人間は本能的に「顔」を見てしまう。
    * **目線**: 被写体の目線が見ている方向に、視聴者の目線も誘導される。文字を見せたいなら、その文字を見つめる写真を使う。
    * **表情**: 「驚き😲」「恐怖😱」「爆笑🤣」など、感情が伝わる表情はクリック率UP。

## 🎨 色とフォントの選び方

### 配色のコツ

* **3色以内**: ごちゃごちゃさせない。
* **補色（反対色）**: 「青背景に黄色文字」「赤背景に白文字」など、コントラストを強くする。
* **高彩度**: YouTubeの白背景の中で目立つよう、ビビッドな色を使う（赤、黄、蛍光色など）。

### フォントのコツ

* **ゴシック体一択**: 明朝体は細くてスマホで見にくい。太めのゴシック体（ヒラギノ角ゴ、Noto Sans JP Boldなど）を使う。
* **文字数制限**: **20文字以内**が理想。パッと見で理解できる長さにする。
* **装飾**: 「白フチ」「ドロップシャドウ（影）」をつけて背景から浮き立たせる（袋文字）。

## 📐 構図パターン

| パターン | 特徴 | 向いているジャンル |
| :--- | :--- | :--- |
| **分割構図** | 画面を「Before/After」や「4分割」にする。情報量を出せる。 | 比較、ランキング、解説系 |
| **顔ドアップ** | 画面の半分以上を顔にする。インパクト大。 | エンタメ、やってみた系 |
| **余白活用** | 文字を置くための「無地スペース」を作る。読みやすさ重視。 | ビジネス、教育系 |
| **集中線・効果** | 漫画的な集中線で中央を強調する。 | 衝撃映像、切り抜き |

> [!TIP]
> **スマホサイズで確認しよう**
> 編集画面で大きく見るのではなく、スマホの小さな画面で見て文字が読めるか？画像がわかるか？を確認するのが最重要です。
