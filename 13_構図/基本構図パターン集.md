# 📐 徹底解説！構図の基本とパターン

構図（Composition）は、視覚表現において見る人の視線を誘導し、作品の意図を効果的に伝えるための設計図です。

## 🎯 代表的な構図パターン

| 構図名 | 英語名 | 特徴・効果 | 使用例 |
| :--- | :--- | :--- | :--- |
| **三分割法** | Rule of Thirds | 画面の縦横を3等分した交点に主役を配置。安定感とバランスが良い。 | 風景、人物ポートレート |
| **日の丸構図** | Center Composition | 画面のど真ん中に主役をドーンと置く。力強さ、シンプル、ストレートな視線集中。 | アイドル、強調したい被写体 |
| **三角構図** | Triangle Composition | 底辺が広い三角形を作る配置。どっしりとした安定感、奥行き、高く見せる効果。 | 建物、集合写真、座っている人物 |
| **対角線構図** | Diagonal Composition | 画面を斜めに横切るラインを意識。動き、躍動感、奥行きを生む。 | 走っている人、道路、川 |
| **放射線構図** | Radial Composition | 視線が一点（消失点）に集まる配置。強烈な遠近感と視線誘導効果。 | トンネル、並木道、集中線 |
| **シンメトリー**| Symmetry | 左右対称または上下対称。静寂、美しさ、威厳、神秘性。 | 神社仏閣、水面の反射 |
| **アルファベット**| C / S Curve | 被写体をC字やS字に配置。流れ、優雅さ、リズム感、柔らかさ。 | 女性のポーズ、海岸線、道 |
| **額縁構図** | Frame within a Frame | 窓枠や木々などで主役を囲む。視線誘導、のぞき見感、物語性。 | 窓越しの人物、トンネルの先の風景 |
| **トンネル構図**| Tunnel Composition | 周囲を暗くしたり囲んだりして、中央奥に視線を集める。没入感、ドラマチック。 | 森の中の道、実際のトンネル |

## ✨ 構図選びのコツ

1. **「何を一番見せたいか？」** を決める。
2. その主役を際立たせるための構図を選ぶ（例：力強く見せたいなら**日の丸**、おしゃれに見せたいなら**三分割**、動きを出したいなら**対角線**）。
3. **余白（ネガティブスペース）** を意識する。詰め込みすぎないことが重要。

> [!TIP]
> 迷ったらまずは **「三分割法」** を試してみよう。交点に主役の「目」などを合わせるだけで、グッとプロっぽくなります。
