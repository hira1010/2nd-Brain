# 💰 資産管理システム

Googleスプレッドシートと連携した自動配当金・資産追跡システムです。

## 📊 ファイル構成

```
07_株/
├── 配当金・資産推移.md    # メインダッシュボード
└── sync_assets.ps1        # 同期実行スクリプト
```

## 🔄 使い方

### 自動同期（推奨）

偶数日（2日、4日、6日...）にAntigravityを起動すると、自動で同期が提案されます。

### 手動同期

PowerShellで以下を実行：

```powershell
.\sync_assets.ps1
```

または、Pythonを直接実行：

```powershell
python ..\18_システム\sync_assets.py
```

## 📈 データソース

**Googleスプレッドシート**:

- Sheet ID: `1lnN_z84DLknNWbQX0YWzHiyn5Ea_Hue9TbxQeHSe3HA`
- シート名: `進捗2026`

**同期されるデータ**:

- 月別配当金（1月〜12月）
- 年間合計
- 配当伸び率
- 総資産額

## ⚠️ 注意事項

1. **スプレッドシートのアクセス権限**  
   「リンクを知っている全員が閲覧可能」に設定する必要があります。

2. **依存関係**  
   初回実行時に自動でインストールされます：

   ```
   pip install pandas requests
   ```

3. **ファイル形式**  
   `配当金・資産推移.md` のテーブル構造を手動で変更しないでください。

## 🛠️ トラブルシューティング

### スプレッドシートに接続できない

- 共有設定を確認
- インターネット接続を確認

### 数値が正しく反映されない

- スプレッドシートのセル位置が変わっていないか確認
- `sync_assets.py` の行番号定数を確認

## 📝 カスタマイズ

データ位置を変更する場合は、`18_システム/sync_assets.py` の以下を編集：

```python
YEAR_2026_COLUMN = 10  # K列
MONTH_START_ROW = 2    # 1月は3行目
TOTAL_ROW = 14         # 合計行
```
