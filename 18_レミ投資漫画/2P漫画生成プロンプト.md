# レミの兵法投資 2P漫画生成プロンプト

このプロンプトテンプレートを使えば、「レミの兵法投資リスト101」の任意のTIPから2ページ漫画を自動生成できます。

---

## 📖 使い方

1. **TIPを選ぶ**: リストから番号とタイトルを選択
2. **プロンプトをカスタマイズ**: `{{TIP_NUMBER}}`, `{{TIP_TITLE}}`, `{{TIP_CONTENT}}`を置き換え
3. **ストーリー要素を追加**: 具体的なシーン説明を記入
4. **2回の画像生成**: 1P目と2P目を別々に生成

## 🔧 カスタマイズのコツ

### コマ割りパターンのバリエーション

**1ページ目**:

- パターンA: 大(40%) → 中(30%) → 小小(各15%) ← 基本形
- パターンB: 中中(各30%) → 中中(各20%)
- パターンC: 大(60%) → 小小小(各13%)

**2ページ目**:

- パターンA: 横長大(50%) → 左右分割(各25%) → 横長(25%) ← 基本形
- パターンB: 横長大(40%) → 横長(30%) → 横長(30%)
- パターンC: 大(70%) → 小小(各15%)

### シーン設定のヒント

1. **オフィスシーン**: 説明系TIP向け
2. **カフェ・リビング**: 日常会話風
3. **グラフや図解の背景**: 数字・データ系TIP
4. **過去のビジュアル**: 歴史・由来系TIP
5. **未来の想像図**: テクノロジー系TIP

### キャラクター配置

- **優斗が質問 → レミが回答**: 教育的内容
- **レミが提案 → 優斗が実践**: アクション系
- **二人で議論**: 議論・比較系TIP
- **レミの独白 → 優斗の感想**: 哲学・マインド系

## 🎯 各カテゴリー別推奨アプローチ

### 📊 基礎知識系 (No.1-25)

- **1P**: 悩みの提示 → レミの格言
- **2P**: 比喩的ビジュアル → 教訓 → 決意
- **ビジュアル**: メタファー、心象風景

### 🛡️ 戦略・リスク管理系 (No.51-75)

- **1P**: 失敗例の提示 → 改善策の示唆
- **2P**: 戦略の図解 → ケーススタディ → 成功
- **ビジュアル**: チャート、戦略図、before/after

### 🚀 未来・テクノロジー系 (No.76-101)

- **1P**: 現状の説明 → 未来予測
- **2P**: SF的未来ビジュアル → 投資チャンス → 展望
- **ビジュアル**: 未来都市、ロボット、宇宙など

## ⚙️ 生成時の注意点

1. **テーマボックス**: 必ず1P目の1コマ目右下に配置
2. **2ページで完結**: ストーリーは2P以内に収める
3. **キャラクター一貫性**: プロンプト内のビジュアルアンカーを必ず含める
4. **コマ割りの変化**: 各TIPで同じレイアウトを避ける
5. **日本語精度**: セリフは正確に指定すること
