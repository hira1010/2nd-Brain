# 漫画生成仕様書 (Manga Specifications)

## 🎨 画像生成サイズ

| 項目 | 値 | 備考 |
| :--- | :--- | :--- |
| **幅 (Width)** | **1200 px** | 必須 |
| **高さ (Height)** | **1700 px** | 必須 |
| **アスペクト比** | **12:17** | 縦長 (Portrait) |

## ✅ 統合ツール (MangaTool.ps1)

これ1つですべての作業が完結します。

`.\MangaTool.ps1` を実行するとメニューが表示されます：

1. **📜 プロンプトを表示** (Noから抽出)
2. **🖼️ 画像をリサイズ** (1200x1700に変換)
3. **📖 見開きプレビュー作成** (2枚を結合して表示)
