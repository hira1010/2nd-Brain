# 📘 漫画制作システム仕様書 (Manga Production Specs)

本ドキュメントは、漫画制作プロジェクトにおける画像生成設定および運用ツールの仕様をまとめたものです。

## 1. 🖼️ 画像生成仕様 (Image Specs)

AI画像生成における基本フォーマット設定です。

| 項目 (Item) | 設定値 (Value) | 備考 (Notes) |
| :--- | :--- | :--- |
| **幅 (Width)** | `1200 px` | 固定サイズ |
| **高さ (Height)** | `1697 px` | AI Studio最適化サイズ |
| **アスペクト比** | `1200:1697` | 縦長 (Portrait) |

> **Note**: アスペクト比が維持されていない画像は、後述のツールで自動リサイズ可能です。

## 2. 🛠️ 統合運用ツール (MangaTool.ps1)

制作フローの効率化のため、統合スクリプト **`MangaTool.ps1`** を使用します。

### 使用方法

PowerShellで以下のコマンドを実行してください。

```powershell
.\MangaTool.ps1
```

### 機能一覧 (Menu)

1. **📜 プロンプト表示 (Show Prompts)**
   - 指定した話数のプロンプトファイルを読み込み、コピー可能な形式で表示します。
2. **🖼️ 画像リサイズ (Resize Images)**
   - フォルダ内の画像を `1200x1697` に一括変換します。
3. **📖 見開きプレビュー (Create Spread)**
   - 2枚のページ画像を結合し、見開き状態での確認用画像を生成します。

## 4. 🎭 演出・脚本ガイドライン (Creative Guidelines)

読者を飽きさせないための重要な演出ルールです。

### 📐 コマ割り変形 (Dynamic Layouts)

単調さを避けるため、以下の変則レイアウトを積極的に採用します。

- **斜めカット (Diagonal Cuts)**: アクションや衝撃的なシーンでコマを斜めに割る。
- **タチキリ (Full Bleed)**: ページ端まで絵を描き、壮大さを表現。
- **コマからはみ出す (Break Out)**: キャラや効果音がコマ枠を突き破る演出。
- **変形コマ (Irregular Shapes)**: 精神世界や不安なシーンで歪んだコマを使用。

### 🔮 キャラクター性格とギャグパターン (Character & Gag Patterns)

#### 💃 レミ (Remi)

クールで知的だが、**3割の確率で「変なスイッチ」が入る**。

##### A. スピリチュアル・モード (Spiritual)

- **口癖**: 「宇宙の法則が…」「波動が低いわね」「前世のカルマが…」
- **優斗の反応**: 「（出たよ…スピリチュアル…）」「（話半分で聞こう…）」

##### B. 超セレブ・モード (Ultra Rich)

- **特徴**: 金銭感覚がバグっている。「1億円？お小遣い程度ね」「島を一つ買えばいいじゃない」
- **優斗の反応**: 「（金持ちの感覚こわっ…）」「（桁が違うんだよなぁ…）」

##### C. 軍師・戦闘モード (Warlord)

- **特徴**: 投資を「血で血を洗う戦い」として語る。急に劇画調になる。
- **優斗の反応**: 「（投資の話だよね…？）」「（なんでそんな物騒なの？）」

##### D. ポエム・芸術家モード (Poetic)

- **特徴**: 急に窓の外を見て詩を詠み始める。「風が…呼んでいるわ…相場の風が…」
- **優斗の反応**: 「（急に浸り始めた…）」「（無視して進めよう）」

#### 🎓 優斗 (Yuto)

- **基本**: 常識人・ツッコミ役。
- **心の声**: レミのボケに対し、**声には出さず心の中で冷静にツッコミを入れる**のが基本スタイル。
