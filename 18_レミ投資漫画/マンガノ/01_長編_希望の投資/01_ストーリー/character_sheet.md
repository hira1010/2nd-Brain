# キャラクター設定資料 (Visual Locks)

ユーザー提供の三面図（Sanmenzu）に基づく厳密なビジュアル定義。
全エピソードのプロンプトでこの定義を使用すること。

---

## メインキャラクター

### 1. レミ (Remi)

**役割**: 謎の美女 / 投資の導き手
**特徴**: クールで知的、少しミステリアス。

**Visual Lock Prompt**

```markdown
(Silver Long Hair, Hime-cut),
(Sharp Red Eyes, Cool Beauty Face),
(Crimson Red Suit Jacket),
(Black Lace High-neck Camisole),
(Tight Red Skirt),
(Black Pantyhose),
(Black High Heels)
```

**Negative Prompt (必須)**

```markdown
white shirt, collared shirt, blouse, school uniform, glasses
```

---

### 2. 優斗 (Yuto)

**役割**: 主人公 / 疲れたサラリーマン
**特徴**: どこにでもいる若手社員。将来的には成長するが、序盤は絶望や焦りがメイン。

**Visual Lock Prompt**

```markdown
(Short Black Hair, Slightly messy),
(Navy Blue Suit),
(White Dress Shirt),
(Blue Necktie),
(Brown Leather Shoes)
```

**Negative Prompt (必須)**

```markdown
beard, mustache, old man, muscular, glasses
```
