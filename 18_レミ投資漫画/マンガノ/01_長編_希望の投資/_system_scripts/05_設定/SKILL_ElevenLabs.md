# Eleven Labs Skill (マンガ・ナレーション・ボイススキル)

## 概要

本プロジェクト『優斗の投資旅路』のナレーションおよびキャラクターボイスを Eleven Labs API を使用して高品質に生成するための設定指針です。

## キャラクター・ボイスプロファイル (Voice Profiles)

### 1. レミ (Remi)

- **声質**: 上品で知的な大人の女性。説得力と神秘性を兼ね備えた響き。
- **推奨設定**:
  - **Stability**: 50%
  - **Similarity**: 80% (一貫性を重視)
  - **Style Exaggeration**: 10%
- **指示文（Prompt Example）**: "冷静かつ包容力のある、高貴な女性の声で。一言一句を丁寧に、相手を導くように。"

### 2. 優斗 (Yuto)

- **声質**: 20代〜30代の誠実な日本人サラリーマン。物語と共に成長する変化。
- **推奨設定**:
  - **Stability**: 45%
  - **Similarity**: 75%
  - **Style Exaggeration**: 25% (感情の変化を出しやすく)
- **指示文（Prompt Example）**: "最初は少し気弱だが、次第に希望と自信に満ちていく若者の声で。リアリティのある日常的なトランク。"

## 生成用設定 (API Configuration)

- **Model ID**: `eleven_multilingual_v2` (日本語による高品質な合成に最適)
- **出力形式**: 44100Hz, 128kbps (ナレーション品質)

## 運用方法

1. 各エピソードのセリフ・ト書きを抽出。
2. キャラクター名に応じてボイス設定を選択。
3. 文脈に応じて「間（ポーズ）」を句読点で調整し、APIへリクエスト。
