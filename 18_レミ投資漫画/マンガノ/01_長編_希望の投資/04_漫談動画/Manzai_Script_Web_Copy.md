# Eleven Labs Webサイト用：コピー＆ペースト台本

APIを使用せず、Webサイト（Speech Synthesis）で音声を生成するためのテキストです。
キャラクターごとにまとめてあるので、設定を切り替えずに連続して生成できます。

---

## 🌹 レミ（Remi）用

**設定**: Model: `eleven_multilingual_v2` / Voice: `Antoni` or `Bella` / Stability: 50%

**▼以下のテキストをコピーして生成してください（パッセージごとに区切るか、Project機能推奨）**

そして、相場の神に愛されすぎて、お札をシュレッダーにかけてるレミです。

ねぇ優斗、最短で億りたいなら、まず寝ている間に利益が1万倍になる魔法の壺を……

あら。じゃあ、全財産を『レミの今日のランチ予想』にレバレッジ1000倍で賭けるのは？

……冗談よ。本当の最短ルートは、ランチ代を削って毎日100円ずつ、世界経済に苗木を植えること。

そう。その100円の苗木に、毎日愛情を込めた『波動』を込めて、深夜に裸で踊るの。

ふふ。時間を味方にし、複利の魔法を信じる者だけが、最後に自由という果実を得るのよ。

「「ありがとうございましたー！」」
