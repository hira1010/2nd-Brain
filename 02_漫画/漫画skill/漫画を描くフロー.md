# 🎨 Manga Master Skill

このスキルは、あなたが生み出す「漫画」の世界観と品質を最高レベルに引き上げ、特に**絵柄の一貫性（安定したキャラクター生成）**と**魅力的なストーリー展開**をAI技術で支援します。

## When to use this skill (使いどころ)
- **Situation**: 新しい漫画案を練りたい時、AI生成でキャラクターの顔が崩れて困っている時、特定の絵柄（レトロ、SF、ビジネス等）を再現したい時。
- **Benefit**: キャラクターが毎コマ別人になる現象を防ぎ、プロ級の安定した漫画表現を自動化できます。

## Knowledge & Resources
このスキルは、以下の核心的なノウハウに基づいています：
- [AI漫画絵柄固定術](file:///c:/Users/hirak/Desktop/2nd-Brain/02_漫画/skill/resources/AI漫画絵柄固定術.md): 絵柄とキャラを固定する8つの黄金律。
- **キャラクター詳細**: `02_漫画/キャラ設定/` 内の各プロファイル。

## Workflow
1.  **Step 1: Format Activation**:
    - ユーザーが `漫画.md` または `4コマ.md` を提示した時点で形式を確定する。
2.  **Step 2: Character Selection**:
    - 番号を提示してキャラクターを選んでもらう。
3.  **Step 3: Art Style Selection**:
    - 画風の選択肢を **100個** ([Art_Style_100.md](file:///c:/Users/hirak/Desktop/2nd-Brain/02_漫画/skill/resources/Art_Style_100.md)) 番号付きで提示し、選んでもらう。
4.  **Step 4: Page Count (Manga Only)**:
    - 通常の漫画形式の場合のみ、ページ数を確認する。
5.  **Step 5: Theme & Generation**:
    - 内容を確認し、`02_漫画/アーカイブ/` へ構成案を保存・プロンプト生成。
    - **Technical Specs**: 画像は必ず **「カラー (Full Color)」**、サイズは **「1000x1414 pixels (A4 ratio)」** を厳守する。
    - **Language Rule**: プロンプト内で指定するセリフ（Speech bubble text）や効果音（SFX）は **「日本語 (Japanese Kanji/Kana)」** で記述する。
    - `02_漫画/ 漫画.md` または `02_漫画/ 4コマ.md` のテンプレートを使用する。
    - **Automatic Archiving**: 生成した構成案は必ず **`02_漫画/アーカイブ/`** に保存する。
    - **Structured Prompting**: プロントを構造化（YAML/Markdown）し、AIが理解しやすい形式で出力する。
    - **Visual Fixation**: キャラクターの外見を文字で長々と説明せず、参照画像に任せる「引き算のプロンプト」を徹底する。
3.  **Consistency Check**:
    - 複数ページの生成時は、必ず「一歩前のページ」の内容を参照（リーディング）させるプロンプトを生成する。

## Usage Instructions
### User Request Example
> "レミと優斗の新しい4コマの内容を考えて"
> "この絵柄をAIに完璧にコピーさせたい"
> "プロンプトを構造化して出力して"

### Execution Steps (Agent)
1.  **Consult Knowledge**: `AI漫画絵柄固定術.md` を開き、今回のタスクに最適な「コツ」を選択する。
2.  **Refine Prompt**: 選択したコツ（キャラシート活用、前のページ参照など）をプロンプト案に組み込む。
3.  **Deliver Solution**: 構造化されたプロンプトや、具体的な生成戦略を提示する。

## Tips
- **Stability First**: 派手なポーズよりも、まずは「顔の安定」を最優先したプロンプトを組むことが成功の鍵。
- **Character Bonds**: レミ（クール実業家）と優斗（真面目学生）の対照的な性格が引き立つコミカルなやり取りをストーリーに盛り込む。
